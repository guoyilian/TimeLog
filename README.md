# TimeLog

简洁高效的Android计时应用，让你的时间可视化。

## 主要功能

### ⏱️ 灵活的计时功能
- 支持自定义计时名称
- 内置常用标签快速选择
- 开始/暂停/停止一键控制
- 实时显示计时进度

### 📊 多维度数据统计
- **日视图**：折线图展示当天24小时的时间分布，动态Y轴自适应
- **周视图**：柱状图展示周内每天的时长对比，点击柱子跳转到日视图
- **月视图**：热力图展示整月的计时情况，点击日期查看详情并可跳转日视图
- **年视图**：12个月热力图概览，点击月份跳转到月视图

### 📝 完善的记录管理
- 查看、编辑、删除历史记录
- 支持添加新记录
- 各项数据可视化且有交互跳转功能
  
### 💾 本地数据持久化
- 使用SharedPreferences安全存储
- JSON格式保存，便于导出备份

## 技术特点
- 纯Java开发
- 自定义Canvas绘制图表
- Fragment架构
- 响应式布局适配
