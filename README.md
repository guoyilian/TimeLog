# TimeLog - Android 计时应用

## 想法来源
每当我想做一件事情并且想要强制保持专注时，我会采用计时的方式，但是由于市场上的App如番茄ToDo，小习惯，Forest等“打开计时”功能的步骤或者设置有点多，不能立即执行计时，请注意我的重点是“立即”，即我想要的效果是一打开APP页面就可以点击计时，因此我多次放弃使用这些成熟的App。（在此我并不否认这些App极为成熟、丰富，且丝滑的功能，它们记录了许多人努力的足迹和保持专注带来的成就感）。而后我便使用手机自带的闹钟秒表计时，但是秒表计时并没有历史记录，我无法查看我的计时记录，我不满足于这一点。所以结合上述两方面的需求，我选择做减法，在现有App给我的灵感的基础上，设计了这款简单的计时软件。希望有人可以用得到它！至少我需要哈哈😉（做自己喜欢、感兴趣、带点小小未知，但又毫无负担的事情就很有动力！纯粹的动力。如果你有一个想法，请抓住它！）

一款简洁美观的 Android 计时应用，支持计时记录、数据统计和多维度数据分析。

## 功能特性

### 🕐 计时功能
- **开始/暂停/停止计时**：支持完整的计时控制流程
- **计时名称自定义**：为每次计时添加自定义名称标签
- **实时显示**：动态显示计时时长
- **数据持久化**：自动保存所有计时记录到本地数据库

### 📊 数据统计
提供四个维度的数据统计分析：

#### 日统计
- 折线图展示当日各时段计时时长分布
- 列表展示当日所有计时记录
- 支持点击计时名称进行**修改**（修改名称或时间）
- 支持**左滑删除**计时记录
- 支持手动**添加计时记录**
- 点击折线图数据点查看详细饼图分布

#### 周统计
- 柱状图展示本周每日计时时长
- 列表展示本周所有计时记录
- 点击柱状图查看当日饼图分布
- 周统计标签展示各任务总时长（响应式流式布局）

#### 月统计
- 热力图展示本月每日计时强度
- 颜色深浅表示计时时长
- 点击热力图格子查看当日详细记录
- 月统计标签展示各任务总时长（响应式流式布局）

#### 年统计
- 热力图展示全年每日计时情况
- 按月份分组展示
- 统计总计时天数和总时长

### 🎨 界面设计
- Material Design 设计风格
- 清新的绿色主题配色
- 流畅的动画过渡效果
- 底部弹窗交互设计
- 涟漪点击反馈效果
- 柱状图点击交互动画

## 技术栈

- **开发语言**：Java
- **最低 SDK**：Android 7.0 (API 24)
- **目标 SDK**：Android 14 (API 34)
- **架构组件**：AndroidX Fragment
- **UI 组件**：Material Design Components
- **数据存储**：SharedPreferences + JSON (Gson)
- **自定义视图**：Canvas 绑制图表

## 环境要求

- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Gradle 8.9

## 安装与运行

### 方式一：直接安装 APK
1. 在 [Releases](../../releases) 页面下载最新的 APK 文件
2. 在 Android 设备上安装 APK（需开启"允许安装未知来源应用"）

### 方式二：从源码编译

1. **克隆仓库**
   ```bash
   git clone https://github.com/guoyilian/TimeLog1.0.0.git
   cd timer
   ```

2. **用 Android Studio 打开项目**
   - 打开 Android Studio
   - 选择 "Open an Existing Project"
   - 选择项目根目录

3. **同步 Gradle**
   - 等待 Gradle Sync 完成
   - 如有网络问题，可配置国内镜像源

4. **运行应用**
   - 连接 Android 设备或启动模拟器
   - 点击 Run 按钮 (Shift+F10)

## 使用说明

### 计时操作
1. 点击底部"计时"标签进入计时页面
2. 输入计时名称（可选，默认为"REC"）
3. 点击"开始"按钮开始计时
4. 计时过程中可点击"暂停"暂停计时
5. 点击"停止"结束计时并保存记录

### 查看统计
1. 点击底部"统计"标签进入统计页面
2. 切换日/周/月/年标签查看不同维度数据

### 管理记录
- **修改记录**：在日统计页面，点击计时记录的名称标签
- **删除记录**：在日统计页面，向左滑动记录，点击删除按钮
- **添加记录**：在日统计页面，点击右下角"+"按钮

## 数据存储

应用数据存储在应用私有目录：
- 路径：`/data/data/com.example.timer/shared_prefs/`
- 格式：JSON
- 内容：计时记录列表（名称、开始时间、结束时间、时长）
